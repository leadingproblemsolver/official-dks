# Decision Kill-Switch: Logic Core v2.4

## Product Overview
Decision Kill-Switch is a high-order analytical instrument designed to falsify assumptions and identify terminal risks in human intent before they manifest. It acts as a cognitive filter for high-stakes decisions, moving beyond simple pros/cons and into the realm of **Logic_Symmetry** and **Assumption_Falsification**.

## Core Philosophies
1. **Surgical Precision**: No fluff. No motivational filler. Only clinical, actionable verdicts.
2. **Assumption Falsification**: Instead of validating why an idea will work, the system identifies exactly what state must be true to break the decision entirely.
3. **Cognitive Reframing**: Tailored perspectives provided in both technical (Precise) and concrete (Regular) linguistic modes.
4. **Frictionless Delivery**: Minimalist, dark-ops aesthetic optimized for high-speed intent extraction.

## Tactical Features
- **Binary Verdicts**: Proceed, Pause (requires more data), or Kill (high probability of failure).
- **Relatable Perspective Adaptive**: 2-3 lines of tailored cognitive reframing.
- **Secondary Nuances**: Three surgical sub-points with strict **ONLY_DO_IF** metrics/deliverables.
- **Shareable Reports**: Direct-link public report portals and downloadable PDF summaries.

## Technical Architecture (Logic Core)
- **Framework**: Next.js 15+ (App Router).
- **Intelligence**: Gemini 1.5 Pro via `@google/genai`.
- **Infrastructure**: Supabase (PostgreSQL + Auth).
- **Styling**: Tailwind CSS v4 / motion (animations).
- **Persistence**: Relational schema with RLS enforced profile quotas (5 generations per user).

## Target User
Engineers, founders, and high-frequency decision-makers who require a clinical, adversarial second opinion to prevent catastrophic resource misallocation.
